Kolumna is a little tool to help designers set the layout on their Sketch designs.

Set your layout on this tool, then go to Sketch and open the Layout Settings with ⌃L. Copy the values shown at the bottom of the tool in the settings panel.

Psst! Some tips:

Click on the values at the bottom to copy them to the clipboard.

Click on a device icon to set the canvas size to that device size.
